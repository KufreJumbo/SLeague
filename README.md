# Scholars League

Scholars League is a self-contained, AI-powered academic platform for K-12 students. It helps students study curriculum-aligned content, take automatically marked quizzes and tests, receive AI-guided support, and compete on real-time class leaderboards.

The first curriculum target is Nigeria's NERDC structure, with the platform designed so additional countries can be added as separate curriculum modules.

## Core Student Flow

1. Student signs up and selects country, grade, and class.
2. The platform loads the correct curriculum, subjects, terms, and topics.
3. Student studies topics, asks the AI tutor questions, and takes quizzes or tests.
4. Assessments are auto-marked and scores update XP, rank, streaks, and badges.
5. The system recommends the next best topic using performance and weak-area signals.

## Main Product Areas

- Home dashboard with daily study tasks, current rank, XP progress, streaks, and next actions.
- Subject hub with term topics, completion state, scores, and recommended practice.
- AI tutor chat scoped to each subject and curriculum level.
- Leaderboard for class, subject, overall, and challenge rankings.
- Performance insights with progress charts, weak topics, and ranking prediction.
- Challenges arena for weekly curriculum-aligned competitions.

## MVP Direction

The first version should focus on a polished Nigeria-first web app:

- Student authentication and onboarding.
- Nigeria curriculum module for selected primary, junior secondary, and senior secondary levels.
- Subject and topic browsing.
- Quiz/test engine with auto-marking.
- XP, streaks, tiers, and leaderboard logic.
- AI tutor placeholder or API-backed tutor.
- Basic performance insights and study recommendations.
